---

Requirements:
 - [Go](https://go.dev/dl/)
 - [Elm](https://guide.elm-lang.org/install/elm.html)
 - [Node.js](https://nodejs.org/)

You need to install NPM Dependencies with: `npm install`.

## Elm-Review:
`npm run review`

## Testing:
`npm run test`

## Developing
`npm run dev`

## Generating
`npm run generate`

## Building
`npm run build`
